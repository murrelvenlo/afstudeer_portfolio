Doelstelling

Net zoals bij het vak Datamanipulatie, heb ik bij dit onderdeel kunnen bewijzen dat ik een probleem kan analyseren en oplossen door middel van een zelfontwikkelde WPF-applicatie middels het MVVM-principe. De applicatie is bedoeld voor een bibliotheek die als hoofdoel het uitlenen van boeken heeft.

Realisatie

Dit project is gebasseerd op C# Entity framework.

Voor dit project heb ik in tegenstelling tot het project van Datamanipulatie, Code-First geprogrammeerd. Verder kwam er een belangrijk onderdeel bij, Generics. Dit maakte de herbruikbaarheid van de Create, Read, Update -en Delete-methoden mogelijk.
